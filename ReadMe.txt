Oxi Development Instructions

You'll need to grab our UE5 Engine binaries in order to develop for Oxi.

Video Instructions Here:  https://youtu.be/uJ_u_f6Ebr4?si=hAZahhSFWilzxtVW

Manual Instructions:

1. Download the zipped up engine here: https://www.benny-wilson.com/LivingInAmerica/UE5_PLA.zip

2. Extract this in the same folder this ReadMe file is in

3. Run the RunProjectBrowser.bat batch file found in this directory

4. When the Project Browser opens up, click the "Recent Projects" tab in the left panel then browse to Oxi.uproject

5. Wait until the editor opens!  The editor can take several minutes to open the first time you launch.
